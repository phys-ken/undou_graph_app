# 運動グラフ設問作成ツール

高校物理基礎の運動グラフ（x-t / v-t / a-t）に関する設問を、教員が手描き入力から自動生成するためのツール。ブラウザUIと、AIエージェントによる一括作問向けの REST API の二つの利用経路を持つ。

## Language

**誤答グラフ（distractor グラフ）**:
グラフ選択肢問題における不正解の選択肢として提示される運動グラフ。本ツールは誤答グラフの生成ロジックを持たず、API 呼び出し側（エージェント）が誤答グラフの仕様を指定する——本ツールは「正答との整合・整形・出力」のみを担当する。
_Avoid_: ダミーグラフ、ノイズグラフ

**グラフ選択肢問題**:
複数の運動グラフ（正答1つ＋誤答グラフ複数）から正しいものを選ばせる設問。教育上の理由（生徒に手描きさせたい）から UI には実装せず、API からのみ生成できる。
_Avoid_: 選択式グラフ問題、グラフ多肢選択
