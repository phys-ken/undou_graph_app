# undou_graph_app — 運動グラフ設問作成ツール

高校物理基礎（等速直線運動・等加速度直線運動）向けの、x-t / v-t / a-t グラフ設問を
さっと自動生成するツールです。ブラウザ上で v-t（または x-t）グラフを手描きするだけで、
対応する他のグラフが自動生成され、設問・解答を PNG / PDF / DOCX / ZIP で出力できます。

`legacy_nami_app`（波の重ね合わせ問題作成ソフト）をベースにフォークしています。

## ブラウザで使う

```bash
python server.py
# → http://localhost:8000 が自動的にブラウザで開く
```

## REST API（AIエージェント連携）

```bash
npm install
npm start
# → :8000 静的UI / :8001 REST API
```

## 主な機能

- v-t / x-t グラフの手描き編集（頂点クリック&ドラッグ）
- 対応する x-t / v-t / a-t グラフの自動導出
- 設問タイプ：グラフ変換型、数値・記述型（加速度・総変位・向き・運動の説明）
  - グラフ選択肢型（誤答グラフから正答を選ばせる設問）は教育上の理由でUIには持たず、REST API からのみ生成可能
- 出力形式：PNG / PDF / DOCX（編集可能）/ ZIP一括
- 白黒印刷対応スタイル（既定）＋ カラーモード（領域の塗りつぶし表示に対応）
- REST API（`POST /api/generate` 等）により、AIエージェントによる一括作問にも対応
